--------------------
ALL GRADLE COMMANDS
--------------------
cd android

# Clean
.\gradlew clean

# Debug APK
.\gradlew assembleDebug

# Release APK
.\gradlew assembleRelease

# AAB for Play Store
.\gradlew bundleRelease

# Install on device
.\gradlew installRelease

# List all tasks
.\gradlew tasks


--------------------
INSTALL APK
--------------------
# Via ADB
adb install cashiee-release.apk

# Or copy to phone and install manually


--------------------
TROUBLESHOOTING
--------------------
Problem: "gradlew not found"
Fix: cd android

Problem: "ANDROID_HOME not set"
Fix: setx ANDROID_HOME "C:\Users\YourName\AppData\Local\Android\Sdk"

Problem: "Signing error"
Fix: See BUILD_WITH_GRADLE.md for keystore setup

Problem: Build too slow
Fix: Add to android\gradle.properties:
     org.gradle.daemon=true
     org.gradle.parallel=true
